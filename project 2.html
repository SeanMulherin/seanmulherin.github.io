To answer that question, we first must take a quick tangent to discuss a popular metric called Mean Squared Error (MSE). Simply put, it tests how inaccurate a model is. Eye-balling graphs can only get us so far, so we need some reliable metrics to quantify accuracy - enter MSE. Remember how the training data set is comprised of the first 80% of days, well what happened to the remaining 20% of days?? We call that remainind set the "test set" and to compute the MSE of a model, we ask our models to PREDICT the remaining 20% of data and then COMPARE that PREDICTION to the ACTUAL test set. So anyways, this is the metric we'll use to quantify how inaccurate our models are. Considering the table below, we are interested in the model that gives us the lowest error (unless you have something nefarious going on and want the highest error for some reason.)
